# XuéBàn 学伴 — Chinese Learning Companion

> **Speak. Transcribe. Learn.**
> An iOS app for learning Mandarin Chinese through voice transcription, AI-powered quizzes, stroke-order animation, and spaced flashcard practice.

---

## Features

### Translate Tab — Voice & Text Input
Record yourself speaking Chinese or type a word/phrase directly. The app:
- Transcribes audio into Chinese characters (voice mode) or looks up typed input (text mode)
- Breaks down the sentence **word-by-word** with English and Pinyin for each word
- Lets you **Accept & Save** the result, which translates the full sentence and stores the vocabulary for use in Quizzes and Flashcards
- Tap any word to open a **stroke-order animation sheet** showing how each character is drawn

### Exam Tab — AI Oral Examination Simulator
Simulate a real Chinese oral exam with an AI examiner:
- **Random** — surprise topic generated by the AI
- **By Lesson** — select a specific lesson from your ingested course material; the exam focuses on that lesson's vocabulary while accounting for all prior material
- **Custom Topic** — enter any subject (e.g. "food", "travel", "measure words")
- The AI examiner asks all questions **in Chinese only**, adapts to your answers, and avoids translation tasks
- Your spoken answer is transcribed and evaluated in real time
- At the end you receive an **overall score (0–100), strengths, areas to improve, and per-question feedback**

### Characters Tab — Character Lookup
Search by English word, Pinyin, or Chinese characters to get:
- Pinyin pronunciation with tone marks
- English meaning
- Animated **stroke-order diagrams** rendered locally from the bundled hanzi-writer dataset — no internet required for animations

### Flash Cards Tab — Vocabulary Practice
Test your recall of saved vocabulary:
- A random card is drawn from your saved word list
- Type the **English meaning or Pinyin** (tone marks optional — diacritic-insensitive matching)
- **Correct** → green banner, move to the next card
- **Incorrect** → red banner showing the correct English and Pinyin; retry the card, reveal the answer, or skip
- **Show Answer** → reveals the answer for review

### Settings Tab — Language
Switch the app UI between:
- English
- Српски (ћирилица) — Serbian Cyrillic
- Srpski (latinica) — Serbian Latin

All labels, instructions, and UI text update instantly. The selection is persisted across app launches.

## Tech Stack

### iOS App

| Layer | Technology |
|---|---|
| UI | SwiftUI (iOS 17+) |
| Audio recording | AVFoundation |
| Text-to-speech | AVSpeechSynthesizer |
| Stroke animations | hanzi-writer (local bundle, served via `WKURLSchemeHandler`) |
| Network | URLSession |
| Localisation | Custom `Strings` struct + `LanguageManager` (English / Serbian Cyrillic / Serbian Latin) |

### Backend

| Concern | Service |
|---|---|
| Framework | FastAPI (Python) |
| Database | PostgreSQL + pgvector (hosted on Render) |
| Chat / LLM | DeepSeek (`deepseek-chat`) via plain httpx |
| Embeddings (RAG) | Jina AI (`jina-embeddings-v3`, 1024 dims) via plain httpx |
| Audio transcription | Groq (`whisper-large-v3-turbo`) via plain httpx |
| Word segmentation | jieba |
| Deployment | Render |

## Getting Started

### iOS App

1. Clone the repo and open `ChineseVoiceTranslator/ChineseVoiceTranslator.xcodeproj` in Xcode.
2. Set your backend API base URL in `APIClient.swift` (the `base` constant).
3. Build and run on a physical device or simulator (iOS 17+).
4. Grant microphone permission when prompted.

### Backend (local)

```bash
# Install dependencies
uv sync

# Copy and fill in environment variables
cp .env.example .env

# Run the server
uv run uvicorn main:app --reload
```

### Ingesting Course Material

```bash
# First-time setup — drop existing table if switching embedding models
# (run in your Postgres DB): DROP TABLE IF EXISTS course_chunks CASCADE;

# Ingest PDF lesson files
python ingest.py pdfs/*.pdf

# Re-ingest a specific file
python ingest.py pdfs/lesson1.pdf --force

# Clear everything and re-ingest all
python ingest.py pdfs/*.pdf --clear-all
```
